## 摘要
目的：年龄相关性黄斑变性和视网膜色素变性等疾病会导致光感受器层退化。恢复视力的一种方法是通过微电极阵列（如视网膜上植入物）电刺激存活的视网膜神经节细胞。已知视网膜上植入物会产生可见的各向异性形状，这些形状沿邻近视网膜神经节细胞的轴突束伸长。最近的研究表明，为了获得各向同性的像素状形状，可以通过使电极失活或降低刺激电流水平来映射轴突束并避免刺激它们。避免轴突束刺激旨在去除画笔描边状形状，转而使用更简化的像素状形状。方法：在本研究中，我们提出使用各向同性和各向异性形状，在名为rlretina的强化学习环境中为虚拟患者的视网膜渲染可理解的图像。该环境将任务形式化为在基于笔触的渲染任务中使用笔触。主要结果：我们训练了一个深度强化学习智能体，该智能体学会组装各向同性和各向异性形状以形成图像。我们研究了哪些基于误差或基于感知的指标足以奖励智能体。该智能体以基于模型的数据生成方式训练，使用心理物理学验证的轴突束映射模型来渲染不同虚拟患者感知到的图像。我们表明，与不同虚拟患者中的朴素方法相比，该智能体可以生成更可理解的图像。意义：这项工作提供了一种解决视网膜上电刺激的新方法，是朝着使用各向异性光幻视改善人工恢复视力中视觉敏锐度的第一步。

## Abstract
Objective: Diseases such as age-related macular degeneration and retinitis pigmentosa cause the degradation of the photoreceptor layer. One approach to restore vision is to electrically stimulate the surviving retinal ganglion cells with a microelectrode array such as epiretinal implants. Epiretinal implants are known to generate visible anisotropic shapes elongated along the axon fascicles of neighboring retinal ganglion cells. Recent work has demonstrated that to obtain isotropic pixel-like shapes, it is possible to map axon fascicles and avoid stimulating them by inactivating electrodes or lowering stimulation current levels. Avoiding axon fascicle stimulation aims to remove brushstroke-like shapes in favor of a more reduced set of pixel-like shapes. Approach: In this study, we propose the use of isotropic and anisotropic shapes to render intelligible images on the retina of a virtual patient in a reinforcement learning environment named rlretina. The environment formalizes the task as using brushstrokes in a stroke-based rendering task. Main Results: We train a deep reinforcement learning agent that learns to assemble isotropic and anisotropic shapes to form an image. We investigate which error-based or perception-based metrics is adequate to reward the agent. The agent is trained in a model-based data generation fashion using the psychophysically validated axon map model to render images as perceived by different virtual patients. We show that the agent can generate more intelligible images compared to the naive method in different virtual patients. Significance: This work shares a new way to address epiretinal stimulation that constitutes a first step towards improving visual acuity in artificially-restored vision using anisotropic phosphenes.

- **研究动机**：传统方法（如朴素刺激算法NSA）试图通过避免刺激轴突束来获得各向同性光幻视，但这限制了可用电极和刺激强度，降低了形状多样性。本文提出**利用**各向异性形状（而非回避），以类似“笔画绘制”（stroke-based rendering, SBR）的思路，将各向同性与各向异性光幻视作为画笔，通过强化学习智能体自动组合这些形状，生成更可理解的视觉感知。
- **整体含义**：首次将基于模型的深度强化学习应用于视网膜植入物刺激，为利用各向异性光幻视改善人工视觉敏锐度提供了新范式。

## 2. 方法论：核心思想、关键技术细节、算法流程

### 核心思想
- 将视网膜上植入物刺激问题形式化为一个**基于笔触的渲染任务**：智能体（agent）在rlretina环境中，通过选择单电极刺激（动作at），逐步叠加各向同性和各向异性光幻视，使累积的感知图像P尽可能接近目标图像I。
- 采用**基于模型的数据生成**方式：环境中的虚拟患者感知由心理物理学验证的“轴突映射模型”（axon map model）模拟，该模型能精确生成与真实患者感知一致的各向异性形状。

### 关键技术细节
- **环境（rlretina）**：基于OpenAI Gym规范构建，模拟Argus II植入物（10×6电极阵列）。单电极刺激产生两种光幻视：
  - 各向同性（scoreboard模型）：$I_{\text{score}}(x,y;\rho) = \exp\left(-\frac{(x-x_{\text{stim}})^2+(y-y_{\text{stim}})^2}{2\rho^2}\right)$
  - 各向异性（axon模型）：$I_{\text{axon}}(x,y;\lambda) = \exp\left(-\frac{(x-x_{\text{soma}})^2+(y-y_{\text{soma}})^2}{2\lambda^2}\right)$
  - 最终感知：$I_{\text{map}} = I_{\text{score}} \times I_{\text{axon}}$（$\rho$和$\lambda$为空间衰减常数，可调以模拟不同患者）
- **智能体**：采用**Soft Actor-Critic (SAC)** 算法，属于深度强化学习中的actor-critic方法。actor学习策略$\pi(a_t|s_t)$，critic估计Q函数。SAC通过熵最大化目标实现更稳定的探索。
- **状态与奖励**：
  - 状态$s_t$由目标图像I和当前感知P拼接组成。
  - 奖励$r(s_t,a_t) = \frac{L_{t+1} - L_t}{L_{t0}}$，其中$L$为I与P之间的距离度量。论文比较了**l2距离**和**Wasserstein距离**作为$L$的估计。
- **网络结构**：actor和critic均使用带CoordConv层的卷积神经网络（CNN），超参数见论文表1（学习率3e-4，折扣因子0.99，缓冲区1e6等）。

### 算法流程（文字描述）
1. 初始化SAC智能体和环境（rlretina）。
2. 每轮episode开始，从MNIST数据集随机选择一张目标图像I。
3. 智能体每次选择动作at（即刺激某个电极），环境根据轴突映射模型生成新的感知P并累加。
4. 计算当前步的奖赏（基于Wasserstein距离的差异变化）。
5. 存储经验进入回放缓冲区。
6. 从缓冲区采样小批量，更新actor和critic网络。
7. 重复以上步骤直至达到设定的步数N（即单电极刺激次数），完成一个episode。

## 6. 主要结论与发现
- **奖赏设计关键**：使用l2距离作为奖赏的智能体很快陷入“模式崩溃”，仅生成低空间频率的模糊团块；而Wasserstein距离奖赏能提供更有效的梯度信息，使智能体学会利用图像的高频细节，生成轮廓清晰的数字。
- **性能对比**：在所有虚拟患者条件下，SAC智能体生成的感知图像在MSSIM指标上显著优于NSA（p<0.0001），尽管在l2和MSE上SAC略差（因为SAC会在高亮度区域外也放置光幻视，增加了像素差异）。
- **适应性**：当植入物远离视网膜（高ρ，大光幻视）且刺激次数增多时，NSA生成的图像几乎不可读，而SAC智能体仍能保持数字可辨别性。
- **意义**：证明了利用各向异性光幻视进行基于笔触的渲染是可行的，且DRL智能体能自动适应不同患者参数，无需手动调参。

## 7. 优点
- **新颖性**：首次将各向异性光幻视视为可利用的“画笔”，而非避之不及的缺陷，为视网膜植入物刺激提供了全新视角。
- **方法学严谨**：使用心理物理学验证的轴突映射模型，且环境、代码开源（rlretina），可复现性强。
- **奖赏设计洞察**：揭示了在感知任务中像素级误差（l2）的局限性，并推荐使用概率分布距离（Wasserstein）作为更合适的奖赏信号。
- **实用性导向**：智能体可自动适应不同虚拟患者（不同ρ和λ），减少了对人工调参的依赖，向临床个体化治疗迈进一步。
