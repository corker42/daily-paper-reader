## 摘要
小样本图像识别要求模型能从少量标记的支持集中适应新类别。赫布快速权重记忆能在一次任务中提供临时关联信息，但固定的记忆行为可能不适合每个小样本任务。在这项工作中，我们提出了面向小样本视觉Transformer的自适应赫布路由。该方法使用轻量级MLP路由器来控制赫布记忆的贡献、记忆更新的强度以及从支持集特征中保留先前记忆的程度。我们研究了自适应放置、自适应可塑性和完全自适应赫布路由。实验在Omniglot、CIFAR-FS以及从CIFAR-FS到Omniglot的跨域迁移上，使用ViT-Small、DeiT-Small和Swin-Tiny进行5-way 1-shot评估。在直接的Swin比较中，固定和自适应赫布变体使用相同的记忆位置。自适应可塑性将固定赫布的结果从96.74%提高到96.92%，而完全自适应路由达到了最好的结果96.94%。与固定赫布Swin相比，完全自适应的Swin模型还将推理时间从16.51毫秒减少到14.05毫秒。在CIFAR-FS上，自适应变体在所有三种骨干网络上均提升了性能，并且多次评估显示，随着支持示例数量的增加，这些增益仍然有效。这些结果表明，自适应可塑性和自适应记忆激活能够改善小样本Transformer表示，超越固定赫布行为。

## Abstract
Few-shot image recognition requires models to adapt to new classes from a small labeled support set. Hebbian fast-weight memory can provide temporary associative information during an episode, but fixed memory behavior may not be appropriate for every few-shot task. In this work, we propose Adaptive Hebbian Routing for few-shot Vision Transformers. The method uses a lightweight MLP router to control the contribution of Hebbian memory, the strength of memory updates, and the retention of previous memory from support-set features. We study Adaptive Placement, Adaptive Plasticity, and Fully Adaptive Hebbian Routing. Experiments use ViT-Small, DeiT-Small, and Swin-Tiny under 5-way 1-shot evaluation on Omniglot, CIFAR-FS, and cross-domain transfer from CIFAR-FS to Omniglot. In the direct Swin comparison, fixed and adaptive Hebbian variants use the same memory location. Adaptive Plasticity improves the fixed Hebbian result from 96.74\% to 96.92\%, while Fully Adaptive Routing achieves the best result at 96.94\%. The fully adaptive Swin model also reduces inference time from 16.51 ms to 14.05 ms relative to fixed Hebbian Swin. On CIFAR-FS, adaptive variants improve performance across all three backbones, and the multi-shot evaluation shows that these gains remain useful as the number of support examples increases. These results show that adaptive plasticity and adaptive memory activation can improve few-shot Transformer representations beyond fixed Hebbian behavior.

## 1. 论文的核心问题与整体含义（研究动机和背景）

- **核心问题**：小样本图像识别要求模型从极少量标注样本（支持集）快速适应新类别。现有Vision Transformers（ViT、DeiT、Swin）作为骨干网络时，其权重在推理阶段固定，难以针对每个任务动态调整。赫布快速权重记忆（Hebbian fast-weight memory）可在一次任务中提供临时关联信息，但**固定不变**的记忆行为（如激活强度、更新速率、衰减率）无法适应不同任务的需求，甚至可能损害性能。
- **研究动机**：探索是否可以根据当前支持集特征，自适应地控制赫布记忆的参与程度、写入强度和保留程度，从而提升小样本学习的表示能力，同时保持计算效率。
- **整体含义**：记忆模块的效用不仅取决于插入位置，还取决于其行为是否随任务动态调整。自适应机制能显著改善小样本Transfomer的适应性和效率。

## 2. 论文提出的方法论：核心思想、关键技术细节

### 2.1 核心思想

- 在Vision Transformer中插入轻量级MLP路由器，接收选定骨干位置的支持集特征（经均值池化），输出三个控制值：
  - **外激活门控** $g_l$：控制赫布记忆输出对当前表示的贡献强度。
  - **可塑性率** $\eta_l$：控制支持集特征写入记忆矩阵的速率（范围为 $[0, \eta_{\max}]$，$\eta_{\max}=0.15$）。
  - **记忆衰减因子** $\lambda_l$：控制前一记忆的保留程度。
- 通过这三个自适应参数，赫布记忆更新变为：
  $$ M_l \leftarrow \lambda_l M_l + \eta_l A_l $$
  记忆增强表示为：
  $$ \tilde{X}_l = X_l + g_l \cdot H_l(X_l, M_l) $$
  其中 $H_l$ 为赫布记忆读出，$A_l$ 为关联矩阵（由键-值外积经裁剪得到）。

### 2.2 关键技术细节

- **路由器结构**：两层MLP，隐藏维度为输入维度的1/8（最小16），输出层初始化使初始门控低、可塑性中等、衰减高，避免初期大更新。
- **记忆重置**：每个任务开始时重置记忆矩阵，确保只存储当前支持集的临时关联。
- **三种自适应变体**：
  - **Adaptive Placement**：仅学习 $g_l$，固定 $\eta, \lambda$。
  - **Adaptive Plasticity**：学习 $\eta_l,\lambda_l$，固定 $g_l=1$。
  - **Fully Adaptive**：同时学习 $g_l,\eta_l,\lambda_l$。

### 2.3 算法流程（文字说明）

1. 对于每个小样本任务，将支持集图像输入Transformer骨干，在选定位置提取特征（通过mean pooling聚合为单向量）。
2. 路由器根据该向量预测三个控制值。
3. 支持集特征写入赫布记忆矩阵（受可塑性率和衰减因子控制）。
4. 支持集和查询集特征在后续层中读取该记忆，输出受门控加权的残差。
5. 所有查询样本通过原型网络（Prototypical Networks）分类。
